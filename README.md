# i4_hackathon

YYZ delay contagion and gate optimization pipeline with dashboard and web app.

## Project layout

```
├── data/
│   ├── raw/              # Input CSVs (yyz_flights.csv, flights_sample_3m.csv)
│   ├── processed/        # Pipeline outputs (merged_flights, delays, beta_scores, gate_assignments)
│   └── figures/          # Plots and contagion_network.html
├── scripts/              # Pipeline: merge → predict → beta → optimize → embed_data
├── lib/                  # Frontend libs (vis-network, tom-select)
├── dashboard.py          # Streamlit dashboard (run from root)
├── server.py             # Flask app for web UI (run from root)
├── index.html            # Web app
├── data.js               # Generated by scripts/embed_data.py
└── requirements.txt
```

## Quick start

1. **Pipeline** (from project root):
   ```bash
   python scripts/0_merge_data.py
   python scripts/1_predict_delays.py
   python scripts/2_compute_beta.py
   python scripts/3_optimize_gates.py
   python scripts/embed_data.py
   ```

2. **Dashboard**: `streamlit run dashboard.py`

3. **Web app**: `python server.py` → http://localhost:8080
